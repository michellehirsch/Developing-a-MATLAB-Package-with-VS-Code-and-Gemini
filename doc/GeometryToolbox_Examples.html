Geometry Toolbox Examples

This page provides links to example scripts demonstrating the usage of the Geometry Toolbox functions.

The example scripts (.m files) can be found in the examples directory of the toolbox.

  • ex_shapes_area.m: Demonstrates area calculations for all shapes.
  • ex_shapes_visualization.m: Demonstrates visualization of all shapes.
  • ex_combined_demo.m: A comprehensive demo showing multiple features.

It is recommended to run these scripts in MATLAB to understand the toolbox capabilities.
