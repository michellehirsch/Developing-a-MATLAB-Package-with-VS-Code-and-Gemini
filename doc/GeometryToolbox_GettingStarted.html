Getting Started with Geometry Toolbox

Welcome to the Geometry Toolbox! This toolbox provides a set of functions for calculating the area of common 2D shapes and for visualizing them.

Installation

To use this toolbox, ensure the root folder GeometryToolbox (containing +geometry, doc, etc.) is on the MATLAB path. You can add it using addpath(genpath('path/to/GeometryToolbox')) or via the MATLAB Set Path dialog.

If installed as an Add-On, MATLAB should handle path management automatically.

Basic Usage

All functions are part of the geometry package. To call a function, use the syntax geometry.functionName().


    radius = 5;
    circleArea = geometry.areaCircle(radius);
    fprintf('The area is: %f\n', circleArea);

    figure;
    geometry.visualizeCircle(radius, [1,1]);
    title('My Circle');

Further Help

For detailed help on specific functions, type help geometry.functionName in the MATLAB command window (e.g., help geometry.areaCircle).
